# 🚀 AI Career Mentor Platform – Intelligent Career Growth & Job Success Ecosystem

A next-generation AI-powered SaaS platform that acts as a complete digital career coach, helping students, fresh graduates, professionals, career changers, and job seekers achieve their dream careers through personalized AI guidance, skill development, interview preparation, resume optimization, and intelligent career planning.

---

## 📌 Problem Statement

Students and professionals often struggle to identify the right career path, understand industry skill requirements, build ATS-friendly resumes, prepare for interviews, track certifications, improve portfolios, and manage job applications. Existing career platforms provide isolated solutions such as resume builders, learning platforms, or job portals, forcing users to switch between multiple applications and making career planning fragmented, time-consuming, and inefficient.

---

## 💡 Solution Overview

**AI Career Mentor Platform** is an all-in-one AI-powered career development ecosystem that provides personalized career guidance using artificial intelligence.

The platform analyzes a user's education, experience, skills, resume, certifications, portfolio, GitHub profile, LinkedIn profile, and career goals to generate customized recommendations that help them become job-ready.

The system intelligently:

- Analyzes resumes using AI
- Calculates ATS compatibility
- Detects skill gaps
- Generates personalized career roadmaps
- Recommends learning resources
- Suggests projects and certifications
- Conducts AI mock interviews
- Tracks coding progress
- Manages job applications
- Monitors career growth through analytics

Instead of relying on multiple career websites, users receive a unified AI-powered experience that continuously evolves with their progress and helps them achieve long-term career success.

# ▶️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/appu-shoo/AI-Career-Mentor.git

cd AI-Career-Mentor
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Configure Environment Variables

Create a `.env.local` file and add:

```env
DATABASE_URL=
OPENAI_API_KEY=
GEMINI_API_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLOUDINARY_URL=
REDIS_URL=
```

---

### 4️⃣ Run Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
